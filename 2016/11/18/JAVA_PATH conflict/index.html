JAVA_PATH conflict

when we have different JAVA_PATH in our computer ,may be there is some problem when run eclipse or other tool that depend on java envrionment.

Example :

At first installed java version 1.8 and config eclipse build path,and install HP QTP/UFT , when run eclipse there is a wrong message “ Picked up JAVA_TOOL_OPTIONS: -agentlib:jvmhook “

Solve method

delete variable

user variable
_JAVA_OPTIONS
-Xrunjvmhook -Xbootclasspath/a:F:\MYSOFT~1\HP\QTP\bin\JAVA_S~1\classes;F:\MYSOFT~1\HP\QTP\bin\JAVA_S~1\classes\jasmine.jar

system environment

_JAVA_OPTIONS
-Xrunjvmhook -Xbootclasspath/a:F:\MYSOFT~1\HP\QTP\bin\JAVA_S~1\classes;F:\MYSOFT~1\HP\QTP\bin\JAVA_S~1\classes\jasmine.jar
