Linux deploy .net application

Install Software:

Install Mono

download

wget http://download.mono-project.com/sources/mono/mono-3.0.12.tar.bz2

cd mono-3.0.12

./configure –prefix=/usr

make

sudo make install

cd ..

Install libuv

sudo apt-get install automake libtool curl

curl -sSL https://github.com/libuv/libuv/archive/v1.4.2.tar.gz | sudo tar zxfv - -C /usr/local/src

cd /usr/local/src/libuv-1.4.2

sudo sh autogen.sha

sudo ./configure

sudo make

sudo make install

sudo rm -rf /usr/local/src/libuv-1.4.2 && cd ~/

sudo ldconfig

Intall DotNet Version Manager,

curl -sSL https://raw.githubusercontent.com/aspnet/Home/dev/dnvminstall.sh | DNX_BRANCH=dev sh && source ~/.dnx/dnvm/dnvm.sh

source /home/parallels/.dnx/dnvm/dnvm.sh

dnvm

dnvm upgrade

Install jexus

download wget http://www.linuxdot.net/down/jexus-5.8.1.tar.gz

tar -zxvf jexus-5.8.1.tar.gz

cd jexus-5.8.1

sudo ./install

jexus start stop restart

　1 start sudo /usr/jexus/jws start

　2 stop sudo /usr/jexus/jws stop

　3 restart sudo /usr/jexus/jws restart

　　or you can just start sitename

　　　　f.1 start sudo /usr/jexus/jws start website(sitename)

　　　　f.2 stop sudo /usr/jexus/jws stop sizeName(sitename)

　　　　f.3 restart sudo /usr/jexus/jws restart sizeName
(sitename)
